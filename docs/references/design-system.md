# Design System & UI Guidelines

## Language Policy

This project follows a bilingual approach:

**User Interface (UI):**
- All user-facing text should be in **Chinese**
- English abbreviations and standard technical terms are acceptable (e.g., "OK", "Cancel", "API")

**Code & Documentation:**
- All code comments must be in **English**
- All technical documentation must be in **English**
- Git commit messages must follow [Conventional Commits](https://www.conventionalcommits.org/) specification in English
- This ensures internationalization consistency and maintainability
